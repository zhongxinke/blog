# Flex

简易的Flex组件，通过简单地传入属性实现布局

## Usage

```ts
   <script setup lang="ts">
   import Flex from '@zxk/components/Flex'
   </script>

   <template>
    <Flex direction="column" justify="center" align="center">
        <div>item</div>
        <div>item</div>
    </Flex>
   </template>
```
